#🚀 ExpenseMate AI

Smart Food, Travel & Expense Recommendation System

#📌 Overview

ExpenseMate AI is a Python-based recommendation system that combines restaurant discovery, travel planning, and expense management into one platform. It uses structured datasets and user preferences to generate personalized recommendations.

#🎯 Objective

To simplify decision-making by providing a single system for food, travel, and budget planning instead of using multiple apps.

#💡 Features
Restaurant recommendations
Travel destination suggestions
Budget-based filtering
Integrated workflow (food + travel + expenses)
Simple web interface
#⚙️ Technology Used
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Data: CSV datasets
Logic: Filtering and ranking algorithms
#🧠 Working

The system takes user preferences such as budget and interests, processes the data from CSV files, and generates ranked recommendations which are displayed through a web interface.

#👥 End Users
Travelers
Food lovers
Budget-conscious users
#📌 Conclusion

ExpenseMate AI provides an easy and efficient way to plan food, travel, and expenses in one place, making it a useful tool for everyday decision-making
